# Sales Data Analysis Dashboard

## Project Overview
This project analyzes sales data to extract business insights using SQL and Power BI. The dashboard provides a clear understanding of sales trends, top-performing products, and regional performance.

## Tools Used
- SQL (MySQL)
- Excel (Data Cleaning)
- Power BI (Visualization)

## Key Insights
- Identified top-performing regions and products
- Analyzed monthly sales trends
- Evaluated customer segments and purchasing behavior

## Dashboard Features
- KPI Cards (Total Sales, Orders, Customers)
- Monthly Sales Trend
- Region-wise Performance
- Top Products Analysis
- Category & Segment Insights
- Interactive Filters (Slicers)
